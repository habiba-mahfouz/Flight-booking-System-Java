# ✈️ Flight Booking System

A robust and user-friendly Java-based application for managing flight reservations. This system offers both a modern Graphical User Interface (GUI) and a classic Command Line Interface (CLI) to cater to different user preferences.

## 👥 Team Project
This project was developed as a **collaborative team effort**. It demonstrates the practical application of core Data Structures and Object-Oriented Programming (OOP) principles in a real-world scenario.

---

## 🌟 Key Features

### 🔐 Multi-Access Authentication
*   **Secure Signup & Login:** User registration system with validation for email and phone numbers.
*   **Persistent Profiles:** Basic account management with profile completion checks for essential travel documents (National ID, Passport).

### 🎫 Booking & Seat Management
*   **Flexible Routes:** Choose between multiple departure and destination cities (e.g., Cairo, Alexandria, Aswan to London, Paris, Dubai).
*   **Tiered Classes:** Support for Economy and First Class bookings.
*   **Smart Seat Allocation:** Uses a **Queue-based** search system to manage seat availability and reservations in real-time.

### 🍽️ Premium Services
Enhance the travel experience with optional add-ons:
*   In-flight Meals
*   High-speed WiFi
*   Comfort Seating options
*   Entertainment packages

### 💳 Payment & History
*   **Dual Payment Modes:** Support for both Cash and Visa Card payments.
*   **Flight Logs:** A dedicated history feature to track all previous bookings, including costs and selected services.

### 🎨 Modern GUI
*   Crafted with **Java Swing**, featuring a professional blue-themed interface with smooth transitions and intuitive navigation.

## 🛠️ Built With

*   **Language:** Java
*   **Framework:** Java Swing (for GUI)
*   **Data Structures:**
    *   **Singly Linked List:** For managing historical flight data.
    *   **Queue:** For efficient seat search and booking logic.
*   **IDE:** NetBeans / VS Code

## 🚀 How to Run

1.  **GUI Version (Recommended):**
    Run `FlightBookingGUI.java` to experience the full interactive interface.
    
2.  **CLI Version:**
    Run `DSProject.java` for a console-based interaction.

---

## 📂 Project Structure

- `src/dsproject/`: Contains all source files.
- `FlightBookingGUI.java`: Main GUI controller and layout.
- `DSProject.java`: Main entry point and CLI controller.
- `Registration.java`: Handles user authentication and data management.
- `Search.java` & `Queue.java`: Core logic for seat reservation.
- `Single_LL.java`: Data structure for flight history.
- `Passenger.java`: Data model for user profiles.
- `BuyTickets.java`, `Payment.java`, `Service.java`: Logic for booking, billing, and add-ons.
